# CSCE106-Advanced

There are two primary reasons you might use this repository:
- You have learned Python before and need less instruction and more advanced exercises
- You have learned Python before and want to complete the course requirements early

## Instructions
1. Clone the repository
    - Google it, if you don't know how to do that
    - I recommend cloning it with VS Code and then working from your local copy
    - Do NOT send me an invitation to view your cloned repo. I will only grade what is in Blackboard.
2. Complete the assignments in order
    - There are comments in each file with detailed instructions
    - Read the directions carefully. Many "advanced" students have earned poor grades because they did not follow the instructions
3. Submit the assignments to Blackboard in their corresponding assignment
    - For example, submit your code for Lab 7 to the Blackboard assignment named "Lab 7"
    - In the text submission area in Blackboard, include the word "advanced"

### Questions?
If you have any questions, send me an email. You can find my email address and office information in the syllabus (see Blackboard).

Do not send Python files via email. Outlook blocks Python file attachments. If you need to send code, copy and paste the text (of the code) into an email or change the file extension to ".txt".
